## Overview

This endpoint returns all products available in the e-commerce catalog. It is a public endpoint that does not require authentication.

## Endpoint

```
GET /products
```

## Authentication

No authentication required. This is a public endpoint.

## Response

Returns an array of product objects.

<ResponseField name="products" type="array">
  Array of product objects
  
  <ResponseField name="id" type="integer">
    Unique identifier for the product
  </ResponseField>
  
  <ResponseField name="name" type="string">
    Product name
  </ResponseField>
  
  <ResponseField name="description" type="string">
    Product description
  </ResponseField>
  
  <ResponseField name="price" type="decimal">
    Product price (decimal with 2 decimal places)
  </ResponseField>
  
  <ResponseField name="imageUrl" type="string">
    URL to the product image
  </ResponseField>
  
  <ResponseField name="stock" type="integer">
    Available stock quantity
  </ResponseField>
  
  <ResponseField name="categoryId" type="integer" optional>
    ID of the category this product belongs to
  </ResponseField>
  
  <ResponseField name="createdAt" type="datetime">
    Timestamp when the product was created
  </ResponseField>
  
  <ResponseField name="updatedAt" type="datetime">
    Timestamp when the product was last updated
  </ResponseField>
</ResponseField>
